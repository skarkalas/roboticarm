# Wiimote Command List
* B button    \-> gripper open
* A button    \-> gripper close
* \+ button   \-> wrist up
* \- button   \-> wrist down
* UP button   \-> shoulder forward
* DOWN button \-> shoulder backward
* 1 button    \-> led on
* 2 button    \-> led off
* flick left  \-> base counter-clockwise
* flick right \-> base clockwise
* flick up    \-> elbow up
* flick down  \-> elbow down
